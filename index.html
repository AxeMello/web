AIM:  
To design  a webpage  using  all HTML  elements.  
ALGORITHM  / DESCRIPTION:  
Step 1: Set up  the basic structure

an HTML document  
Step 2: Add a  header  
Inside  the <body>  element,  add a header  section  using  the <header>  tag. You can 
include a heading, logo, or any other content you want in the header.  
Step 3: Create a  navigation menu  
Below  the header,  add a navigation  section  using  the <nav>  tag. Inside  the <nav>  tag, 
you can use an unordered  list <ul>  and list items  <li> to create  your navigation menu.  
Step 4: Add main content  
After  the navigation,  include  the main  content  of your webpage  inside  the <main>  tag. 
This can include headings, paragraphs, images, videos, and other elements.  
Step 5: Save  and view  your webpage  
Save  your HTML  file with a .html  extension  and open  it in a web browser  to see your 
webpage design.
